# miibo_discord連携bot
miiboとdiscordを連携するbotです。

.node-version、.nvmrcは、ホスティングサービスrenderにて、Node.jsの新しいversionを使用するためのものです。
renderを使用しない場合は削除してください。
- [render.com: Setting Your Node.js Version](https://render.com/docs/node-version)
